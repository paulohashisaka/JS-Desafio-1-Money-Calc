# Money Calc

A simple currency converter developed during my JavaScript studies with DevClub.

## About the project

Money Calc is a currency converter built to practice JavaScript fundamentals, DOM manipulation, functions, events, and basic user interaction.

## Technologies used

- HTML5
- CSS3
- JavaScript

## Features

- Currency selection
- Value conversion
- DOM manipulation
- Interactive button
- Clean layout
